# Al Amreyah Cement Budget Analysis Dashboard

## Overview

Financial performance analysis requires more than reviewing isolated statements. Decision-makers need an integrated view of revenue, expenses, profitability, balance sheet position, and liquidity trends to identify risks early and take corrective action.

This project delivers a complete **interactive Power BI dashboard** for **Al Amreyah Cement**, designed to analyze financial performance across multiple perspectives including **P&L, budget variance, balance sheet position, liquidity, and working capital**.  
The dashboard compares **Actual Current Year**, **Actual Previous Year**, and **Budget Current Year** to uncover performance gaps, highlight structural issues, and support better financial decisions.

## Tools & Technologies Used

- **Power BI Desktop**: For building interactive dashboards and report pages
- **DAX (Data Analysis Expressions)**: For KPIs, variance calculations, liquidity metrics, and financial comparisons
- **Power Query**: For data cleaning, transformation, and preparation
- **Bookmarks & Navigation Buttons**: For seamless report navigation
- **Slicers & Filters**: For dynamic analysis by quarter, month, location, level, and subsidiary
- **Comparative Analysis Framework**: To compare Actual Current Year, Actual Previous Year, and Budget Current Year

## Analytical Highlights

- The analysis revealed a significant deterioration in current-year financial performance, with losses increasing sharply compared to the previous year.  
- Revenue declined materially without a proportional reduction in expenses, creating pressure on profitability.  
- Budget comparison showed substantial negative variances, especially on the revenue side.  
- Liquidity weakened significantly, with a noticeable deterioration in current ratio and working capital.  
- The dashboard helps identify structural issues in working capital management, financial reporting patterns, and year-end performance concentration.

## About This Project

This Power BI project provides an interactive financial analysis of Al Amreyah Cement, covering revenue, expenses, profitability, budget variance, balance sheet position, liquidity, and working capital.
